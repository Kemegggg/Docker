# Лабораторные работы по Docker и контейнеризации

Данный репозиторий содержит материалы для выполнения лабораторных работ по дисциплине  
«Интеграция и развертывание программного обеспечения с использованием контейнеров».

В репозитории представлены:
- пошаговые инструкции по выполнению заданий
- примеры работы с Docker
- Dockerfile и конфигурации
- примеры развертывания приложений
- базовые сценарии работы с контейнерами и облачными средами
